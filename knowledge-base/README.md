# 河海大学高校知识库

本目录用于存放河海大学校园问答助手的知识库文档。

## 当前目录结构

- `demo/`：演示版示例文档，当前包含 2 份示例资料
  - `学校简介.md`：学校概况与基本信息
  - `student-services.md`：学生事务服务示例
- `INDEX.md`：全量知识库索引（仅作归档与后续导入参考，精简演示版不直接参与索引）

## 说明

当前仓库为精简演示版，仅保留 `demo/` 下的少量示例文档，便于快速体验问答效果。
预构建的 Demo 向量库对应上述文档，clone 后下载即可运行，无需重新消耗 Embedding Token。

如需接入完整的河海大学知识库，请将完整知识库文档放回本目录，并重新构建索引：

```bash
cd backend
uv run python -m app.cli
```

> 完整版知识库可覆盖学校概况、组织机构、教育教学、科学研究、招生就业、人才招聘、信息公开、新闻动态等栏目。
