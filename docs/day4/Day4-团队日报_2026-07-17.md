# 校园问答助手 Day4 团队日报

## 今日目标

完成知识库 CRUD、文档异步处理、向量化入库、独立检索和前端管理页面。

## 今日完成

1. 完成 PDF、DOC、DOCX、TXT、Markdown 上传，单文件限制 50MB。
2. 完成后台异步解析、分块、Embedding 与 FAISS 持久化入库。
3. 完成列表、搜索、筛选、分页、状态轮询、预览和下载。
4. 完成标题更新、重新处理、批量处理、批量删除和级联清理。
5. 提供独立 TopK 向量检索接口，返回相似度、文本片段和来源。
6. 完成前端上传进度、错误反馈、空状态和处理阶段展示。

## 验收结果

- 文档上传后能够自动进入处理流程并最终变为可检索状态。
- FAISS 索引能够加载，当前演示索引包含 402 个向量、维度为 1024。
- 删除文档后，数据库记录、原文件和对应向量一并移除。
- 自动测试覆盖上传、处理、预览、下载、检索、更新、重处理和删除链路。

## 风险与衔接

真实向量化依赖有效模型密钥和网络；自动测试使用本地伪 Embedding，避免验收受外部服务波动影响。Day5 将在该检索能力上完成 RAG 问答和历史记录。
